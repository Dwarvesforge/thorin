# t_array_head

Returns all elements in an array except for the last one.


### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$items  |  **{ [Array](http://php.net/manual/en/language.types.array.php) }**  |  The array to process  |  required  |

Return **{ [Array](http://php.net/manual/en/language.types.array.php) }** All the items except the last one

### Example
```php
	Thorin::array_head([1,2,3]);
// [1,2]
```
