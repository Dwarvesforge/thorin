# t_root_url

Return the root website url depending on the T_ROOT_URL



Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$slug  |  **{ String }**  |  An optional slug to add to the root url  |  required  |

Return **{ String }** The corresponding root url
